Projekte

architektur-barthold-klinger.de

Ausgangspunkt

Für ein Architekturbüro soll eine Website mit abgeschlossenen Projekten erstellt werden.

Idee

Ziel ist, Bilder von abgeschlossenen Projekten in eine grafisch schlichte Webseite einzubetten. Einzelne Projekte werden auf Projektseiten mit mehr Bildern und etwas Text näher vorgestellt.

Umsetzung

Grafisch minimalistisch sehen Besucher als erstes eine Collage aller Projekte. Um die mobile Nutzung zu erleichtern, signalisiert ein Icon, dass sich hinter dem Bild eine Projektseite verbirgt.

justabear.de

Ausgangspunkt

Meine Band benötigte eine Webseite. Und ich hatte schon länger die Idee, programmieren zu lernen. Mithilfe eines guten Freundes mit langjähriger Erfahrung begann ich am lebenden Objekt Webentwicklung zu lernen.

Idee

Erfahrungsgemäß interessieren Besucher von Bandwebseiten weder die Haustiere der Bandmitglieder noch ihre Schulbildung. Daher sollte die Musik, Live-Termine und Social Media im Vordergrund stehen.

Umsetzung

Mit html, CSS und bootstrap erstellte ich in Absprache mit der Band einen One-Pager, der mit wenig Scrollen alle relevanten Inhalte und Informationen enthält. Schauen Sie doch mal rein.

haeapua.fi

Ausgangspunkt

Ein befreundeter Entwickler arbeitete an einem eigenen Projekt; einer Freelancer-Plattform, die für verschiedenen Dienstleistungen wie Nachhilfe oder Umzüge Helfer und Auftraggeber zusammenbringt.

Idee

Er bat mich, das Front-End (die Benutzeroberfläche) mit Bootstrap und so wenig CSS wie möglich umzusetzen, um so schnell wie möglich einen funktionsfähigen und anpassbaren MVP vorweisen zu können.

Umsetzung

Alle Views wurden vorwiegend mit Bootstrap umgesetzt. Da das Projekt auf Ruby on Rails basiert, wurden dynamische Elemente und Daten in embedded Ruby umgesetzt.
